# Men's Wear E-Commerce Website

A modern, stylish, and fully responsive Men's Fashion & Clothing website template built entirely with vanilla HTML, CSS, and JavaScript.

## 🚀 Features
- **Responsive Design:** Fully optimized for all devices including mobile, tablets, and desktops.
- **Modern UI/UX:** Clean, elegant, and user-friendly interface designed specifically for a premium men's fashion brand.
- **Smooth Navigation:** Interactive navigation menu and seamless scrolling experience.
- **Interactive Elements:** Dynamic product filtering and hover effects using vanilla JavaScript.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
